\section*{Introducci\'on}

Los números complejos son fundamentales en diversas áreas de la física computacional, como oscilaciones, transformadas de Fourier y mecánica cuántica. Por ello, disponer de herramientas eficientes para operar con ellos es indispensable. Mientras que Python ofrece comodidad de uso, C++ proporciona eficiencia computacional. El objetivo de este trabajo es implementar una clase para números complejos en C++, convertirla en una librería dinámica e integrarla con Python mediante un wrapper, aprovechando así las ventajas de ambos lenguajes.
