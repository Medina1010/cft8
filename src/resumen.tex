\begin{abstract}
  Este trabajo presenta la creación de una librería en C++ diseñada para realizar operaciones básicas con números complejos, incluyendo suma, resta, multiplicación, división, módulo y conjugado. Posteriormente, la librería fue integrada con Python mediante un mecanismo de interoperabilidad, permitiendo utilizar estas funciones desde un entorno más flexible. Se realizaron pruebas en Python que confirmaron su correcto funcionamiento. El proyecto demuestra la eficacia de combinar la rapidez de C++ con la versatilidad de Python en aplicaciones de física computacional.
\end{abstract}
